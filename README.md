# Blood-Donation-Management-System-Full-Stack-Project
Developed a full stack Blood Donation Management System using React.js, Spring Boot, and MySQL to manage donors, blood requests, and hospital records. Implemented REST APIs, CRUD operations, authentication, and responsive UI for efficient donor search and emergency blood request management.

## Features
- Donor Registration
- Blood Request Management
- Search by Blood Group
- REST API Integration
- Responsive UI

## Technologies Used
- React.js
- Spring Boot
- MySQL
- Tailwind CSS

## Project Modules
- User Management
- Donor Management
- Blood Request System
- Admin Dashboard

## Installation
1. Clone the repository
2. Run Spring Boot backend
3. Run React frontend
4. Configure MySQL database
